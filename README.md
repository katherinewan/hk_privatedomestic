# Hong Kong Private Domestic Rental EDA

This project analyzes Hong Kong private domestic rental trends from 2015 to 2025 using exploratory data analysis (EDA).
It compares rental patterns across regions and property classes to understand market changes over time.

## Dataset
Source: data.gov.hk / Rating and Valuation Department 
Period used: Nov 2015 to Nov 2025
Regions: Hong Kong Island, Kowloon, New Territories 
Classes: A to E

## Methods
- Data cleaning
- Univariate analysis
- Bivariate analysis
- Multivariate analysis [file:143]

## Main Finding
The rental market dropped sharply during the pandemic and later entered a recovery phase, while smaller units showed stronger resilience than luxury units. [file:143]
